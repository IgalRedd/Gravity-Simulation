Controls:
When the game is paused you can right click to add a new planet
To add mass by the 10s press U, else press Y to remove mass by the 10s to minimum 10
Right click an existing planet to remove it

When the game is still paused you can left click on an existing planet to select it.
Once selected, you can press a,w,s,d to change a planet's velocity

Finally during any point of the game you can press: i,j,k,l to move the camera
